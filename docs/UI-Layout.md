# Reading Provocateur — UI Layout 설계

> Learning-first PRD 기반 + Clara AI 레퍼런스 + Newsprint 디자인 시스템
> **웹 (Vite + React 19)**, 데스크톱 우선 + 반응형
> 디자인 시스템: `design-system prompt.md`

---

## Clara AI 레퍼런스 — Learning-first 재매핑

Clara AI = "AI로 문서 읽고 보고서 쓰는 도구" (비즈니스/전략)
RP = "AI로 책 읽고 **진짜 이해하는** 도구" (학습/재시도 엔진)

```
Clara AI 파이프라인:
Resources → Summaries → Provocations → Writing

RP 파이프라인:
PDF → Intent Selection → Provocation → Answer + Confidence
  → Evaluation → Retry → Review Items → Export
```

| # | Clara AI 패턴 | RP Learning-first 적용 | Phase |
|---|--------------|----------------------|-------|
| 1 | 하이라이트+메모+AI 논평 동시 | **하이라이트 → Intent → 도발 → 답변 → 평가** 한 플로우 | **P1** |
| 2 | 인라인 AI 주석 버블 | Intent chips (핵심/헷갈림/연결/적용) | **P1** |
| 3 | Text Dimensions 슬라이더 | 세션 모드 선택 (이해/적용/시험/비판) | **P1** |
| 4 | 원본↔인사이트 양방향 링크 | 도발 카드 → 원본 페이지 점프 | **P1** |
| 5 | 로딩 중 구조 프리뷰 | AI 평가 중 로딩 (verdict 대기) | **P1** |
| 6 | OUTLINE 사이드바 | **도발 트레일** — 학습 여정 타임라인 | P2 |
| 7 | 섹션별 요약 카드 그리드 | **챕터 검증** 결과 카드 | P1.5 |
| 8 | AI 생성 대안 프리뷰 | [다른 도발] 재생성 | P2 |
| 9 | 리소스 사이드바 | 라이브러리 뷰 | P3+ |

**핵심 전환**: Clara AI에서는 "요약/인사이트 생성"이 중심이었지만,
RP에서는 **"평가 → 재시도"** 가 중심으로 올라왔다. UI도 이에 맞게 재설계.

## 3. 플로팅 툴바 + Intent Selection

```
                  ┌─────────────────────────────────┐
                  │ 많은 개발자들이 자신의 커리어를     │
                  │ ██████████████████████████████   │
                  │ ██ 소프트웨어 장인정신 없이 ████   │
                  │ ██████████████████████████████   │
                  │                                  │
                  │   ┌────────────────────────────┐ │
                  │   │ [🟡 Highlight] [💭 Provoke] │ │
                  │   └────────────────────────────┘ │

```

**Newsprint 스타일**:
- 툴바: `bg-[#111] text-[#F9F9F7] border-0 px-4 py-2 font-sans text-sm uppercase tracking-wider`
- 호버: 각 버튼 반전 (`hover:bg-[#F9F9F7] hover:text-[#111]`)
- 하이라이트 색: `rgba(254, 243, 199, 0.6)` (warm yellow)

### Step 2: [💭 Provoke] 클릭 시 → Intent Chips 전개

```
                  │ ██████████████████████████████   │
                  │ ██ 소프트웨어 장인정신 없이 ████   │
                  │ ██████████████████████████████   │
                  │                                  │
                  │   ┌────────────────────────────┐ │
                  │   │ 왜 여기서 묻길 원해?        │ │
                  │   │                            │ │
                  │   │ [핵심] [헷갈림] [연결] [적용] │ │
                  │   └────────────────────────────┘ │
```

**Intent Chip 스타일**:
- 칩: `border border-[#111] px-3 py-1.5 font-sans text-xs uppercase tracking-widest`
- 호버: 반전 `hover:bg-[#111] hover:text-[#F9F9F7]`
- 선택: 반전 고정 + 작은 dot indicator

**Intent → 질문 종류 매핑 (시각적 단서)**:

| 칩 | 라벨 아래 힌트 | 생성되는 질문 경향 |
|----|--------------|-------------------|
| 핵심 | "왜 중요해?" | recall, compression |
| 헷갈림 | "뭐가 안 맞아?" | misconception probe |
| 연결 | "뭐랑 닿아?" | connection, transfer |
| 적용 | "어디 써먹어?" | transfer, synthesis |

→ Intent 선택 즉시 사이드 패널 State Loading으로 전환
